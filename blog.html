Question-1: What are the different ways to select an element in the DOM?

Select elements using method such as getElementById(), getElementsByClassName() and querySelector() for single elements or querySelectorAll() for multiple elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The key difference between innerHTML, innerText, and textContent is that innerHTML allows setting or retrieving HTML content, innerText gets or sets text while respecting styles, and textContent retrieves or sets text while ignoring styles and HTML elements.

Question-3: What is event delegation in the DOM?

Event delegation is a way to handle events by placing a single event listener on a parent element instead of adding one to each child. When a child element is clicked, the event travels up to the parent, which can detect and handle it. This makes the code faster and works well for elements added later.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM means an event starts from the target element and moves up to its parent elements, allowing them to handle it too.

Question-5: How do you create, add, and remove elements using JavaScript?

To create, add, and remove elements: createElement, appendChild, removeChild, remove.
